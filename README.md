Mais informações podem ser encontradas em [VSS-SDK][vss-sdk].

Installation
------------
[INSTALL][install]

Cloning
-------
```
$ git clone --recursive https://github.com/SIRLab/VSS-Viewer.git
```

Development and Use
-------------------
```
$ mkdir build && cd build
$ cmake ..
$ make
$ ./VSS-Viewer
```
